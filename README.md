# MetalSlug

    Metal Slug in C and SDL 2

## building:

    ./install_sdl.sh
    git submodule update --init --recursive
    mkdir build
    cd build
    cmake ..
    make

## run:

    ./metalSlug

## usage

    movement:               arrows keys
    jump:                   s
    shoot:                  x
    fullscreen(on/off):     f
    fullscreen(off):        Esc
